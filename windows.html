Chrome Always On Top + Instant Screenshot Tool

Force Chrome to stay on top of EVERYTHING (even fullscreen games/videos) + Ctrl+Shift+S → screenshot to clipboard

PART 1 – MAIN COMPUTER (Run the Python script)

Step 1: Install Python

Download & install Python 3.12 (or latest):

Direct Download Python 3.12.7 (64-bit)

IMPORTANT: During installation → check "Add Python to PATH" → Install for all users → Install

Step 2: Download the Python Script

Download always_on_top_chrome.py

Step 3: Install Required Libraries

Open Command Prompt as Administrator (Right-click Start → Command Prompt (Admin)) and run:

pip install pywin32 pyautogui keyboard psutil --upgrade

Step 4: Run the Script (Always as Admin)

  • Right-click the downloaded always_on_top_chrome.py → "Copy as path"
  • Open Command Prompt as Administrator
  • Paste this and press Enter:
python "C:\path\to\your\always_on_top_chrome.py"

→ Chrome will now stay on top FOREVER, even over fullscreen apps.

→ Press Ctrl+Shift+S anytime → instant screenshot saved to clipboard.

Keep this Command Prompt window OPEN. To stop: press Ctrl+C

PART 2 – VIRTUAL MACHINE (Run the PowerShell script)

Do this ONLY on your VM (not main PC)

Step 1: Download PowerShell Script

Download the .ps1 script

Step 2: Open PowerShell as Administrator

Right-click Start → Windows PowerShell (Admin)

Step 3: Allow Scripts to Run

Set-ExecutionPolicy Unrestricted -Scope CurrentUser -Force

Step 4: Run the Script

Simply drag the downloaded .ps1 file into the PowerShell window → press Enter

Done! You're all set

Main PC: Chrome floats on top + instant screenshots
VM: PowerShell script running

Works perfectly on Windows 10 & 11
