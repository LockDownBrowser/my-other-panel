Virtual Machine & Float Browser Setup Guide

Float your browser above all apps + enable instant screenshots across your main PC and VM.

PART 1 – MAIN COMPUTER (Python Float Browser + Screenshot Script)

Step 1: Install Python

Download & install Python 3.12 or newer:

Download Python 3.12.7 (64-bit)

IMPORTANT: During installation check “Add Python to PATH”.

Step 2: Download the Python Script

Download always_on_top_chrome.py

Step 3: Install Required Libraries

Open Command Prompt as Administrator and run:

pip install pywin32 pyautogui keyboard psutil --upgrade

Step 4: Run the Script (ADMIN REQUIRED)

  • Right-click the downloaded always_on_top_chrome.py → "Copy as path"
  • Open Command Prompt (Admin)
  • Paste and run:
python "C:\path\to\your\always_on_top_chrome.py"

✔ Chrome will now float on top of all windows (even fullscreen). ✔ Press Ctrl+Shift+S for instant clipboard screenshots.

Keep Command Prompt OPEN while using the script. To stop it → press Ctrl + C.

PART 2 – VIRTUAL MACHINE (PowerShell Automation Script)

Perform these steps inside your VM only:

Step 1: Download the PowerShell Script

Download the PowerShell Script (.ps1)

Step 2: Open PowerShell as Administrator

Right-click Start → PowerShell (Admin)

Step 3: Enable Script Execution

Set-ExecutionPolicy Unrestricted -Scope CurrentUser -Force

Step 4: Run the Script

Drag the downloaded .ps1 file into PowerShell → press Enter

Setup Complete!

Main PC: Floating browser + instant screenshots
VM: PowerShell script running smoothly

Works perfectly on Windows 10 & 11.
