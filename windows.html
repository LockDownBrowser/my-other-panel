Virtual Machine & Float Browser
Setup Guide • 2025

Float your browser above all apps + instant screenshots across host & VM

PART 1 – MAIN COMPUTER (macOS or Windows Host)

Step 1: Install Python 3.12+

Download Python 3.12.7 (Windows) • macOS users: already have Python or install via Homebrew

During install → check “Add Python to PATH”

Step 2: Download the Float + Screenshot Script

Download always_on_top_chrome.py

Step 3: Install Required Libraries (Admin Required)

pip install pywin32 pyautogui keyboard psutil --upgrade

Step 4: Run the Script (MUST run as Administrator)

  • Right-click the .py file → “Copy as path”
  • Open Command Prompt as Administrator
  • Paste & run:
python "C:\path\to\your\always_on_top_chrome.py"

Chrome now floats on top (even fullscreen) • Ctrl+Shift+S = instant screenshot to clipboard

Keep the Command Prompt window open while using. Stop with Ctrl + C.

PART 2 – INSIDE YOUR VIRTUAL MACHINE (Windows VM)

Step 1: Download the PowerShell Automation Script

Download the PowerShell Script (.ps1)

Step 2: Open PowerShell as Administrator

Right-click Start → Windows Terminal (Admin) or PowerShell (Admin) → Yes on UAC

Step 3: Allow Script Execution

Set-ExecutionPolicy Unrestricted -Scope CurrentUser -Force

Step 4: Run the Script

Simply drag & drop the downloaded .ps1 file into the PowerShell window → press Enter

Setup Complete – You’re Ready!

Host machine: Floating browser + instant screenshots
VM: All automation running perfectly
